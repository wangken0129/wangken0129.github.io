    Calm:

    ~ or ~

        Control plane VIP
        Ingress VIP
        Load balancing pool

    Prism Element cluster:
        AHV: 20201105.1045 or later
        AOS: 5.19.1 or later

    Project must have monitoring enabled (console)
    A service account (how-to)
        Role: Project Owner
        A private key: JSON format
