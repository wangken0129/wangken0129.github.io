Install OCP

Requirements

  1. Storage Space: 800GB

  2. 1 Cluster VIP for load balancer on control plane

  3. 1 Ingress VIP for load balancer on worker node

  4. DNS record : vip.(cluster name).(base domain) *.apps.(cluster name).(base domain)

  5. IPAM with 7 ip addresses.

  6. Virtual machines:

  • 1 temporary bootstrap node
  • 3 control plane nodes
  • 3 compute machines
  • 1 installation bastion machines
  1. SSH Key
  2. download oc , ccoctl, openshift-install,
  3. CoreOS image on web server or Nutanix Object
  4. Prism Central CA
